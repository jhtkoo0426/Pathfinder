About Pathfinder

Pathfinder is an interactive web application that visualises the path-finding process between railway stations on the London Railway Network.

Motivation

This project is an extension of one of my university courseworks (UCL COMP0005 Algorithms Module), which I built solely to demonstrate and test my skills in developing web applications, as well as applying algorithms I have learnt from the module.

Features

This version supports:

  • Entering the names of 2 stations
  • Visualing the route on the map
  • No selection of path-finding algorithms (To be changed)

Tools Used

Programming Languages: Python, HTML, CSS, JavaScript

Web Framework: Django
